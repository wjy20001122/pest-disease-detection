# 完整任务描述文件（按项目模块）

更新时间：2026-05-03（工作台天气可视化 + QnA 重排 + 统计页去重已完成）

## 1. 用户与认证模块
### 已完成
- 注册、登录、登出、个人资料更新主链路可用。
- 注册页接入邮箱验证码；登录页支持忘记密码与验证码重置流程。
- 前端鉴权、Token 注入、路由守卫、管理员权限控制可用。
- 权限拒绝（401/403）审计日志可查询、可视化。
### 待完成
- 无。

## 2. 检测核心模块（图像/视频/摄像头）
### 已完成
- 图像检测：上传 -> 检测 -> 结果回传 -> MySQL 落库。
- 管理员：图像/视频/摄像头可用；图像支持手选模型直连本地模型。
- 普通用户：仅保留图像检测，强制 `crop_type + category` 约束。
- 视频检测：Celery + Redis 异步任务，状态与结果持久化，支持轮询/停止/回放。
- 视频链路可实时展示处理进度与中间结果，异常快速失败并返回明确提示。
### 待完成
- 无。

## 3. 审查与通知模块
### 已完成
- 高风险检测触发审查，通知入库并支持已读/全读。
- `review_events` 误报事件落库，失败容错不影响主检测结果。
- 审查阈值、区域预警阈值支持后台配置与默认回退。
### 待完成
- 无。
